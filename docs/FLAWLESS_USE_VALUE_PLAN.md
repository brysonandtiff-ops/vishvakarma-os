# Vishvakarma.OS Flawless Use Value Plan

## Target

Vishvakarma.OS is flawless when a user can create or load a project, edit the blueprint, inspect the matching 3D model, save safely, export useful evidence, and reopen the same project without drift.

## Core workflow

1. Create or load project.
2. Edit 2D blueprint.
3. Confirm live 3D matches the 2D state.
4. Save through local mode or Supabase mode.
5. Export project evidence.
6. Reopen and confirm the state is unchanged.

## Value gates

- One canonical project model.
- Safe save, load, fallback, and recovery states.
- Reliable undo and redo for editor mutations.
- Wall-bound doors and windows.
- Accurate measurements.
- Live 2D to 3D sync.
- iPad-safe controls and layout.
- Professional export evidence.
- Evidence-first release gates.

## Next build target

Canonical ProjectModel proof, save/load/recovery tests, and 2D to 3D sync evidence.

Avoid AI, marketplace, collaboration, and extra dashboard expansion until this core workflow is proven.
