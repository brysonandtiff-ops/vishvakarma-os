# No-Drift UI Changelog

## 2026-06-21 — Tooling polish layer

This release pass adds a final presentation-only CSS layer for tooling surfaces.

### What improved

- Tooling controls get calmer press/active/focus feedback.
- Tool rail section labels are more consistent and readable.
- Menu hover/focus states are clearer.
- Canvas empty-state card has stronger visual hierarchy.
- Floating editor HUD/minimap/status surfaces keep their glass treatment.
- Reduced-motion users are protected from extra transitions.

### What did not change

- No route changes.
- No auth wording or auth handler changes.
- No Supabase/database/schema changes.
- No PWA lifecycle changes.
- No editor data model changes.
- No logo/brand source changes.
