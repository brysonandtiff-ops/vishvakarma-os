# Keyboard Shortcuts

## Tools

| Shortcut | Action |
|----------|--------|
| V | Select |
| W | Wall |
| D | Door |
| N | Window |
| M | Measure |
| Shift+M | Dimension |
| T | Text label |
| F | Furniture |
| C | Column |
| U | Stair |

## Selection

| Action | Result |
|--------|--------|
| Click wall | Select wall |
| Shift+click wall | Toggle in multi-selection |
| Drag on empty canvas | Marquee select walls |
| Escape | Cancel marquee |
| Delete / Backspace | Delete selected wall(s) |

## View

| Shortcut | Action |
|----------|--------|
| G | Toggle grid |
| Shift+S | Toggle snap |
| 3 | Toggle 3D view |
| Shift+D | Toggle dimension visibility |

## Edit

| Shortcut | Action |
|----------|--------|
| Ctrl/Cmd+Z | Undo |
| Ctrl/Cmd+Shift+Z or Ctrl/Cmd+Y | Redo |
| Delete / Backspace | Delete selected wall |

Double-click a label in select mode to edit its text inline.
