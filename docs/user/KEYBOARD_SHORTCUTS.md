# Keyboard Shortcuts

## Tools

| Shortcut | Action |
|----------|--------|
| V | Select |
| W | Wall |
| D | Door |
| N | Window |
| M | Measure |
| Shift+M | Dimension |
| T | Text label |
| F | Furniture |

## View

| Shortcut | Action |
|----------|--------|
| G | Toggle grid |
| Shift+S | Toggle snap |
| 3 | Toggle 3D view |
| Shift+D | Toggle dimension visibility |

## Edit

| Shortcut | Action |
|----------|--------|
| Ctrl/Cmd+Z | Undo |
| Ctrl/Cmd+Shift+Z or Ctrl/Cmd+Y | Redo |
| Delete / Backspace | Delete selected wall |

Double-click a label in select mode to edit its text inline.
