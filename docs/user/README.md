## Guided learning

Use the in-app **Tutorial hub** from the editor help button, workspace sidebar, or command palette (`Ctrl+K` / `Cmd+K` → Learn). Start with:

| Tutorial | Best for | Route |
|----------|----------|-------|
| Essentials | First session, wall drawing, 3D preview, Project Proof | `/editor` |
| Your First Floor Plan | Hands-on room shell with openings | `/editor` |
| Labels & Export | Client-ready annotation and deliverables | `/editor` |
| Projects Library | Searching, duplicating, archiving, and reopening projects | `/projects` |
| Design Optimization | Candidate scoring from goals and constraints | `/optimization` |
| Governance OS | Spec locks, change requests, releases, and audit trail | `/spec-center` |

Open any track from the command palette by searching the tutorial name, route, tool, or task.
